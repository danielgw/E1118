These binaries are the compiled from the applications provided in the Xplain
software framework. See the table below for correspondance between binary
file name, application and application configuration.

Binary                          Application         Configuration

display-demo-xplain.hex         display-demo        xplain
display-training-xplain.hex     display-training    xplain
uart-loopback-xplain-bc.hex     uart-loopback       xplain-bc
uart-loopback-xplain.hex        uart-loopback       xplain
xplain-bc-xplain.hex            xplain-bc           xplain-bc

Binaries are generated using the GNU toolchain released with WinAVR 20100110.
